# GitHubSkills

A collection of [Claude Code](https://docs.anthropic.com/en/docs/claude-code) skills for auditing, managing, and improving GitHub repositories.

## Skills

| Skill | Description |
|-------|-------------|
| `repo-health-check` | Audit a repository against quality best practices and produce a scored health report |

## Setup

1. Clone this repository
2. Ensure `gh` CLI is installed and authenticated (`gh auth status`)
3. Open the repository in Claude Code — skills are automatically available

## Usage

Skills are triggered naturally in Claude Code. For example:

- "Run a health check on my repo"
- "Audit phmatray/my-project"
- "Is my repo set up properly?"
